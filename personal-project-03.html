KEY FEATURES
주요 기능

다기능 제품 및 섬세한 사용자 경험 개선 기능을 중요 차별화 요소로 선정

‘다기능 사용자 친화적 오브제 핸드 블렌더 디자인’


DETAILS
상세

키워드에 걸맞는 제품 3가지를 최종 제출

  • 사용자 친화적인 : 탈부착 가능한 텀블러 파츠를 통해 사용자의 외출 준비 시간 단축 및 편의 제공
  • 예술적인 : 본체를 휘감는 듯한 역동적인 디자인, 주방에 예술을 더하는 오브제 핸드블랜더
  • 다기능 : 블랜딩 중 사소한 재료 과열을 시각적으로 나타내는 베이킹 전용 핸드블랜더

    • 팀원 - 김민건, 김인하, 심규진, 어윤구, 홍서윤
